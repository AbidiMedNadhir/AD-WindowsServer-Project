# Windows Server 2019 – Active Directory Administration and Security

This repository contains an academic project developed as part of the Windows Server & Active Directory Security module.

The project focuses on deploying, configuring, and securing an Active Directory environment using Windows Server 2019, following fundamental system administration and cybersecurity practices.

---

## 🎯 Project Overview

The main goal of this project is to understand how Active Directory works in an enterprise environment and how to secure it against common misconfigurations and attack techniques.

---

## 🏗️ Environment

- Domain Controller: Windows Server 2019
- Domain Name: test.local
- Services: Active Directory Domain Services (AD DS), DNS, DHCP
- Client Machine: Windows 10

---

## 🔐 Implemented Features

- Active Directory domain creation and configuration
- Organizational Units (OU) management
- User and group management with password policies
- Group Policy Objects (GPO) configuration for security enforcement
- DHCP service configuration
- Windows Defender Firewall rule configuration
- Backup and recovery using Windows Server Backup

---

## ⚔️ Active Directory Attack Simulation

For educational purposes, the project includes a study and simulation of common Active Directory attacks, including:

- Kerberoasting
- Mimikatz

These simulations were performed in a controlled lab environment to understand Windows authentication weaknesses and improve defensive security knowledge.
